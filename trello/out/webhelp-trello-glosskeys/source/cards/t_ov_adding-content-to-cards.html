Adding content to cards

Trello provides numerous content, classification and communication features on the details view of each card.

  1. Click a card to open its details view.
  2. Add or edit content on the card.
    • If you want to edit the card title, click the title (1).
    • Click Edit the description (2) and type the card description. You can use markdown.
    • Assign members to the card, add labels, add a due date, create an interactive checklist, add attachments (3)
    • Subscribe to the card, if you are not already monitoring it, or vote for the card (4)
    • Write comments and replies to other members (5)
    • Share or print the card (6)
  3. If you want to delete a card, click Archive.
    Items are not deleted, only archived in Trello.
  4. To close the card, click the close (X) button.
